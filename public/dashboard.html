Cómo obtener tu API Key de Gemini

  1. Ingresa a Google AI Studio e inicia sesión con tu cuenta de Google.
  2. Haz clic en el botón azul en la esquina superior izquierda que dice "Get API key" (Obtener clave de API).
  3. Haz clic en el botón "Create API key" (Crear clave de API).
  4. Selecciona o crea un nuevo proyecto de Google Cloud (es 100% gratuito) y haz clic en "Create API key in existing project".
  5. Copia la clave generada (comienza con AIzaSy...) y pégala en el campo de configuración del formulario.
